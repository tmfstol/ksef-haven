# Facturo

Program do faktur KSeF z asystentem AI, kosztorysami budowlanymi, kartami pracy i CRM.

Strona: https://facturo.info

## Stack

- Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (auth, baza, edge functions, storage)

## Uruchomienie lokalne

```sh
npm install
npm run dev
```
